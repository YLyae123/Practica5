¿Que es el Frontend?

El front-end es la parte de una página web o aplicación que interactúa directamente con el usuario. Todo lo que el usuario ve y con lo que interactúa en un navegador web (como botones, imágenes, textos, menús) es parte del front-end. Es la "capa de presentación" de un sitio web, lo que significa que incluye todo el diseño visual y la experiencia de usuario.
